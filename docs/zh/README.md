### vue-core-video-player

<img width="120" src="https://img1.wxzxzj.com/vue-core-video-player-logo@2x.png" />

一款基于 vue.js 的轻量级的视频播放器插件插件

#### Features

+ 个性化配置
+ i18n
+ 服务端渲染
+ 画中画模式
+ 事件订阅
+ 易于开发
+ 移动端适配
